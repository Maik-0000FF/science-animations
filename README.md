# Science Animations

Wissenschaftliche Animationen mit [Manim](https://www.manim.community/) (4K, 60fps).

## Installation

```bash
pip install manim numpy
```

## Verwendung

```bash
manim -qk --fps 60 physics/heating_curve.py HeatingCurveDE
```
